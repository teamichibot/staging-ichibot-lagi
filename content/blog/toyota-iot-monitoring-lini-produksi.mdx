Toyota Manufacturing Indonesia (TMMIN) mengoperasikan ratusan mesin produksi yang bekerja secara terus-menerus. Tantangan utama yang mereka hadapi: tidak ada cara untuk mengetahui kondisi mesin secara real-time tanpa harus menghentikan produksi untuk inspeksi manual.

## Tantangan

Sebelum implementasi, tim maintenance Toyota harus melakukan inspeksi manual secara terjadwal. Artinya, jika ada kerusakan yang berkembang di antara jadwal inspeksi, mereka baru mengetahuinya setelah mesin benar-benar turun — menyebabkan downtime tak terencana yang mahal.

Masalah spesifik yang perlu diselesaikan:

- Tidak ada data real-time kondisi mesin (suhu, getaran, arus listrik)
- Inspeksi manual membutuhkan waktu dan mengganggu jadwal produksi
- Kerusakan baru terdeteksi setelah berdampak pada output produksi

## Solusi yang Kami Implementasikan

Ichibot memasang sistem IoT monitoring di seluruh equipment kritis tanpa menghentikan operasi produksi. Prosesnya:

### 1. Pemasangan Sensor Non-Contact

Kami menggunakan sensor clamp-on untuk arus listrik dan sensor akselerometer untuk getaran yang bisa dipasang tanpa modifikasi pada mesin. Tidak ada downtime untuk instalasi.

### 2. Gateway IoT Lokal

Data dari semua sensor dikumpulkan oleh gateway IoT yang terhubung ke jaringan internal pabrik. Gateway ini juga berfungsi sebagai buffer jika koneksi internet terganggu.

### 3. Dashboard Real-Time

Dashboard berbasis web yang bisa diakses dari browser manapun — PC control room, tablet supervisor, atau HP manajer pabrik. Menampilkan status setiap mesin, tren historis, dan alert otomatis.

### 4. Sistem Alert

Ketika sensor mendeteksi nilai di luar threshold normal, sistem mengirim notifikasi ke tim maintenance sebelum mesin sempat rusak.

## Hasil

Setelah implementasi, tim maintenance Toyota dapat melihat kondisi semua mesin secara real-time dari satu dashboard. Beberapa anomali berhasil terdeteksi dan ditangani sebelum berkembang menjadi kerusakan serius.

Proyek ini terus berjalan dan berkembang — kami secara aktif menambahkan titik monitoring baru sesuai kebutuhan operasional.
