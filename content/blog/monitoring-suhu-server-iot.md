## Pentingnya Suhu di Data Center

Ruang server adalah otak dari pabrik digital. Kenaikan suhu beberapa derajat saja bisa memperpendek umur perangkat atau bahkan menyebabkan *shutdown* mendadak.

### Langkah Implementasi

1. **Penempatan Sensor**: Pasang sensor di titik *hot aisle* (belakang rak) dan *cold aisle* (depan rak).
2. **Setup Gateway**: Hubungkan sensor ke gateway IoT Ichibot menggunakan koneksi WiFi atau Ethernet.
3. **Konfigurasi Alert**: Atur notifikasi WhatsApp jika suhu melebihi 25°C.
4. **Dashboard Visual**: Pantau tren suhu mingguan untuk mengecek efisiensi sistem AC (Cooling).

Dengan monitoring mandiri, Anda bisa tidur tenang tanpa khawatir server pabrik kepanasan.
